Full-Stack Web3 Engineer | Solana Specialist | Developer Relations

Hey there! I'm Matt, a full-stack Web3 engineer with 5+ years building decentralized applications. Currently serving as Developer Relations Engineer at ZetaChain, where I help developers build Universal Apps that work across any blockchain.

I've deployed multiple smart contracts to Solana mainnet handling real economic value, built 6+ Model Context Protocol (MCP) servers enabling AI agents to interact with blockchain ecosystems, and maintain 57+ open-source repositories across three GitHub accounts.

My expertise spans from low-level Rust/Anchor smart contract development to full-stack TypeScript applications with React frontends. I'm particularly passionate about the intersection of AI and blockchain technology.

Currently based in Annecy, France. When not shipping code, I'm exploring the Alps or working on the next cross-chain innovation.
